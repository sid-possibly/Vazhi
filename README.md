# Vazhi (വഴി) 🗺️
**Kerala Transit Intelligence Platform (KTIP)**

Vazhi is a real-time public transit and civic mobility platform for Kerala. It features an interactive, 3D animated map interface designed to visualize transport modes like Metro, Bus, and Water Metro.

## 🛠️ Tech Stack
- **Frontend:** React, Vite, TypeScript, Tailwind CSS
- **3D Graphics:** React Three Fiber (Three.js)
- **Backend:** Node.js, Express (Planned)
- **Database:** PostgreSQL (Planned)

## 📁 Repository Structure
To keep the project organized for the team, we use a monorepo structure:
- `/client`: The Vite + React frontend (3D Map & UI).
- `/server`: The Express API and Database logic.
